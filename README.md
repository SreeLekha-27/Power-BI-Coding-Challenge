# Perfume Sales Analytics Dashboard – Power BI Project

## Description
This project is a complete end-to-end Power BI analytics solution built using a perfume product dataset. It includes data cleaning, modeling, DAX measure creation, visualization, insights generation, and dashboard publishing. The dashboard provides a clear and interactive view of sales, pricing, stock availability, and brand performance.

---

## Dataset Structure

### **Fact Table**
- Perfume Master
- Contains Brand, Type, price, Last Updated Date, Last Updated Time, Gender, Country, Available, and Sold 

### **Dimension Tables**
- **Brand Dimension**  
- **Type Dimension**  

---

##  Key Features

###  Sales & Pricing Insights
- Trend line showing price vs sales pattern  
- Category-level analysis to understand premium segment performance  
- Brand-wise contribution and comparison  

###  Stock & Availability
- Stock levels by brand 
- Indicators identifying products with low stock or supply gaps  

###  Advanced KPI Indicators
- Sales growth %  
- Price performance trend  
- Stock availability KPI  

###  Advanced DAX Measures
- Time intelligence (MTD, QTD, YTD)  
- Last month vs current month comparison  
- Growth % calculation  
- Ranking (Top N perfumes, Top brands, etc.)  

---

##  Tools Used
- **Power BI Desktop**  
- **DAX Measures**  
- Data Modeling 
- **Power Query – Data Cleaning & Transformation**

---

##  Data Modeling Workflow
1. **Data Import & Cleaning**  
2. **Transform data in Power Query**  
3. **Create Fact & Dimension tables**  
4. **Build relationships (Star Schema)**  
5. **Create advanced DAX measures**  
6. **Design visual dashboards**  
7. **Generate insights & recommendations**

---

##  Dashboard Insights
- Perfume sales trend by date  
- Top-performing brands  
- Category price variation  
- Premium vs standard category performance  
- Stock status visualization  
- KPIs for sales growth, price trend & target comparison  

---


##  Conclusion
This perfume analytics dashboard provides a clear understanding of sales trends, brand performance, category pricing, and stock availability.  
With advanced DAX measures and interactive visuals, the project supports data-driven decisions for pricing, promotions, and inventory planning.
